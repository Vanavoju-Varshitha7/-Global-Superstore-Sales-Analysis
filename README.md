# 🛒 Global Superstore Sales Analysis

## 📌 About
End-to-end exploratory data analysis of Global Superstore
sales data using NumPy and Pandas.
Focused on profit analysis, customer segmentation,
discount impact, and business insights.

---

## 📊 Dataset
- **Source:** Kaggle — Global Superstore Dataset
- **Rows:** 51,290 orders
- **Features:** 20 columns including sales, profit,
  discount, segment, market, region

---

## 🎯 Objectives
- Find total sales, profit, and margin using NumPy
- Identify top and bottom performing categories
- Analyze discount impact on profit
- Segment customers by spending value
- Find loss-making products and regions

---

## 📈 Key Insights
- Technology is the highest revenue category
- Home Office segment has highest profit margin (19.68%)
- High discounts (>20%) lead to negative average profit
- 2014 had highest sales and profit among all years
- Strong negative correlation between discount and profit

---

## 🧮 Analysis Performed
- National totals using NumPy (sum, mean, std)
- Category and sub-category profit analysis
- Loss-making order detection using boolean filtering
- Customer segmentation using np.percentile
- Discount vs profit correlation using np.corrcoef
- Year-wise growth using np.diff
- Outlier detection using IQR method
